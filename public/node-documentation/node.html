Node Documentation:

Node.js is an open-source, cross-platform runtime system made for developing server-side webapplications

Node.js is written in Javascript and can compile in node.js runtime on many platforms.

Syntax examples:

  • var
  • let
  • const

3 different kinds of syntax as listed above.

"var" is used as a stanadrd global variable notation, while "let" is a notation used inside a scope and "const" is the notation used when a variable holds a value it is initiated with and therefor not changing, which explains the word "constant" since the value is constant



Strings:

Strings is a key component in many code-langugages and javascript and node.js is no excuse.

Strings can be set to a variable on for example be logged to the console:

let animal = "dog"

console.log("my animal is a ", animal)

Output: my animal is a dog

We can also use template string with backtics as string identifiers. This way we can actually insert variables into text:

let firstname = "Gustav"

let myName = `My name is ${firstName}`

console.log(myName)

Output: My name is Gustav
